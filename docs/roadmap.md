# Roadmap

## Feature Progress

## 一、遊戲流程與核心機制（30%）
- [x] 顯示 15×15 之標準棋盤 （10%）
- [x] 黑子（先行）與白子兩邊輪流下棋於點上 （10%）
- [x] 能判斷結束並顯示贏家 （10%）

## 三、介面與使用者體驗 門檻：需一、二項合計≥50% 才予評分
- [x] 盤面、按鈕、輪到誰等資訊清楚易讀（5%）
- [x] 棋子顏色、外觀顯示美觀（5%）
- [x] 操作過程不卡頓，下棋時有視覺反饋且流暢（5%）
- [x] 有明確的操作提示、贏家提示，誤操作時有適當反饋（5%）

## 四、進階功能 門檻：需一、二項合計≥50% 才予評分
- [x] 支援悔棋功能，需可開關 （5%）
- [x] 支援落子限時功能，需可開關 （5%）
- [ ] 支援儲存/載入棋局（5%）
- [ ] 支援回放系統、結果分享等進階功能（5%）

## 五、加分項目 門檻：需一、二、三項合計≥60% 才予評分（最高給分 20%）
- [ ] 遊戲開始、落子、勝利、失敗有對應音效，無破音、延遲（+5分）
- [x] 內建人工智慧對手，需可開關 （+5分）
- [ ] 支援Network連線對戰功能 （+10分）
- [x] 有做GUI (+10分)
