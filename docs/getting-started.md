# Getting Started

## Prerequisites

- Linux / WSL2
- CMake (後端建置)
- Python 3 與 `venv`

## Backend Setup (C++)

1. 進入 `backend/build`
2. 執行 `make`
3. 如需直接執行後端，可執行 `./gomoku`

## Frontend Setup (Python)

1. 在專案根目錄建立虛擬環境：
   - `python3 -m venv .venv`
2. 啟用虛擬環境：
   - `source .venv/bin/activate`
3. 安裝依賴：
   - `pip install -r requirement.txt`
