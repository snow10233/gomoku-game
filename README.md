# Gomoku Game

使用 C++ 後端引擎與 Python (PySide6) 前端的五子棋專案。

## Quick Start

1. 建立並啟用虛擬環境：`python3 -m venv .venv && source .venv/bin/activate`
2. 安裝 Python 套件：`pip install -r requirement.txt`
3. 建置後端：進入 `backend/build` 後執行 `make`
4. 啟動前端：進入 `frontend` 後執行 `python3 main.py`
